## Запуск проекта

### Скопировать репозиторий
```bash
git clone https://github.com/mMan69/R4C.git
```
### Создать Docker-образ
```bash
make build
```
### Запустить проект
```bash
make up
```

Проект запустится по адресу: http://0.0.0.0:8000/
